# 极速固定风险开仓器

一个为 1 分钟短线交易设计的极简网页工具：
在固定单笔风险（例如 2U）前提下，快速反推建议仓位价值、保证金和可开数量。

## 技术栈
- React + TypeScript + Vite
- Tailwind CSS
- Zustand（含 localStorage 持久化）

## 启动方式
```bash
npm install
npm run dev
```

## 主要功能
- 极速模式（默认）：方向切换、入场价、止损价/止损百分比双模式、风险额快捷按钮、杠杆快捷按钮、实时计算结果。
- 目标位参考：默认展示 1R / 1.5R / 2R / 2.5R，方向切换自动联动。
- 操作区：一键复制交易计划、一键重置。
- 高级模式（折叠）：手续费修正、滑点修正、可用余额校验、自定义 R 值。
- 本地持久化：默认风险额、默认杠杆、修正开关与参数、最近方向、止损模式等自动保存并恢复。
