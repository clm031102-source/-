# 交易中枢（增量优化版）

## 启动
```bash
npm install
npm run dev
```

## 本次系统修正
- 全站视觉重构：主题改为深色克制金质感，统一 token、卡片、按钮、输入、下拉、表格、tooltip。
- 仪表盘重构：新增概览、关键指标、收益简图、盈亏分布、最近记录、快速洞察、快速入口。
- 总结页补全：时间线 + 详情区，支持周期信息、问题与改进重点展示。
- 开仓计算器恢复双层：极速模式 + 高级模式（默认折叠），包含手续费/滑点修正、余额校验、自定义R。
- 统计分析页重构：标题行工作台按钮折叠筛选，图表与明细表联动真实数据。
- 新增交易页支持简洁版/详细版双模式，切换不丢字段，使用同一 Trade 结构保存。

## 路由
- 交易仪表盘
