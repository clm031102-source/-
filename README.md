# 交易复盘系统（优化版）

## 启动
```bash
npm install
npm run dev
```

## 本次重点优化
1. 交易记录“详情”改为浮窗，不再跳页面卡住返回路径。
2. 整体视觉改为更高级深色风格：更强层次、玻璃感面板、导航更醒目。
3. 新增交易中所有关键数字字段增加“怎么填”的说明文案。
4. 新增「策略录入」页面，可维护每套策略的检查清单、入场/出场/风控规则。
5. 统计页支持“全部策略合并”与“按单一策略”切换统计。
6. 新增交易时可直接选择策略。
7. 新增「开仓计算器」菜单，并优化风险/杠杆标签：
   - 前三个默认标签可直接编辑数字
   - 可继续新增标签
   - 新增标签可删除
   - 彻底解决 backspace 删除后残留 0、输入变成 088 的问题（改为字符串输入处理）

## 目录
- `src/pages` 页面
- `src/features` 业务模块
- `src/components/ui` 通用组件
- `src/store` 状态与持久化
- `src/types` 类型定义
- `src/data` 示例数据
