# 交易复盘系统（深色金质感增强版）

## 启动
```bash
npm install
npm run dev
```

## 本次增量优化
- 保留原有交易记录、仪表盘、策略、详情浮窗、计算器等路由与功能。
- 全站视觉升级为深色克制金质感：统一 token、卡片层次、按钮/输入框/下拉框皮肤、表格与 tooltip 风格。
- 统计分析页重构为分析中心：组合筛选、核心指标卡、收益曲线、月度盈亏、策略/品种/方向/情绪/星期维度、盈亏分布、洞察提示、明细表。
- 数据全部基于真实交易记录实时计算，不写死图表数据。

## 主要页面
- 仪表盘
- 交易记录
- 新增交易
- 统计分析
- 策略录入
- 总结
- 开仓计算器
